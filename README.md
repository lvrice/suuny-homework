# suuny-homework 🌈

一份为 **幼儿园中班** 小朋友设计的 **20 天小作业册**，静态网页形式，可浏览、可翻页、可一键打印（A4）。

## ✨ 特性

- 📅 **20 天作业**，每天一页 A4
- 🔢 **四大模块**：
  - 模块一：认识 1-100 数字（每天 10 个，含填空格）
  - 模块二：10 以内加法（每天 10 道，分两列）
  - 模块三：英文字母（每天 3 个大小写，配四线三格）
  - 模块四：汉字书写（每天 3 个，配米字格描红）
- 🎨 儿童友好风格：鲜艳柔和配色、大字号、圆角卡片、童趣图标
- ⬅️➡️ 上一页 / 下一页按钮、快速跳转、方向键翻页
- 🖨 **一键打印**：每页单独打印一张 A4，打印时自动隐藏导航按钮

## 🚀 使用方法

1. 下载仓库后，直接用浏览器打开 `index.html`（推荐 Chrome / Safari / Edge）
2. 使用底部按钮翻页，或使用"快速跳转"下拉菜单直接跳到某一天
3. 想打印某一天：切换到那一天 → 点击 **🖨 打印本页**
4. 地址栏会自动记录 `#day=N`，刷新/收藏都能回到同一天

## 📁 文件结构

```
suuny-homework/
├── index.html    # 页面骨架
├── styles.css    # 儿童友好样式 + 打印样式
├── script.js     # 数据与渲染逻辑
└── README.md
```

## 🧒 内容分配

| 模块 | 20 天内容 |
|------|-----------|
| 数字 | 第 1-10 天顺序写 1-100；第 11-20 天再循环一轮，加强记忆 |
| 加法 | 所有题目之和 ≤ 10，共 200 题，按天分组 |
| 字母 | 每天 3 个大小写字母，覆盖 A-Z 后循环复习 |
| 汉字 | 60 个中班认知范围内的高频简单字（一、二、日、月、人、口、爸、妈 ...） |

## 🎯 小提示

- 打印前可先用浏览器"打印预览"确认版面
- 建议使用 A4 纸，页边距设为"默认"或"无"
- 若想修改字体/颜色/内容，直接编辑 `styles.css` 与 `script.js` 即可
